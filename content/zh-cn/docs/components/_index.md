# 组件

Hugo Saasify提供了丰富的预构建组件库，帮助您快速构建SaaS网站。所有组件都使用TailwindCSS构建，完全可自定义。

## 可用组件

浏览我们的组件库，找到您需要的：

- **[Hero区块](/zh-cn/docs/components/hero/)** - 引人注目的着陆页Hero区块
- **[定价表格](/zh-cn/docs/components/pricing/)** - 多种风格的精美定价展示
- **[功能网格](/zh-cn/docs/components/features/)** - 有效展示您的产品功能

## 使用组件

组件可以在内容文件中通过Hugo短代码使用，或通过创建自定义布局来使用。每个组件都设计为：

- **响应式** - 适用于所有屏幕尺寸
- **无障碍** - 遵循WCAG指南
- **可自定义** - 易于修改样式和内容

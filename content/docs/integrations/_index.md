# Integrations

Hugo Saasify integrates seamlessly with popular tools and services to extend your website's functionality.

## Available Integrations

- **[Analytics](/docs/integrations/analytics/)** - Track visitor behavior and conversions
- **[Forms](/docs/integrations/forms/)** - Collect leads and user feedback

## Setting Up Integrations

Most integrations can be configured in your `hugo.toml` file under the `[params]` section. Each integration page provides specific setup instructions.

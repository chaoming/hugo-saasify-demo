# Components

Hugo Saasify comes with a rich library of pre-built components to help you build your SaaS website quickly. All components are built with TailwindCSS and are fully customizable.

## Available Components

Browse through our component library to find what you need:

- **[Hero Sections](/docs/components/hero/)** - Eye-catching hero sections for your landing pages
- **[Pricing Tables](/docs/components/pricing/)** - Beautiful pricing displays with multiple styles
- **[Feature Grids](/docs/components/features/)** - Showcase your product features effectively

## Using Components

Components can be used in your content files using Hugo shortcodes or by creating custom layouts. Each component is designed to be:

- **Responsive** - Works on all screen sizes
- **Accessible** - Follows WCAG guidelines
- **Customizable** - Easy to modify styles and content

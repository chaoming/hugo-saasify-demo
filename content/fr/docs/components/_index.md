# Composants

Hugo Saasify est livré avec une riche bibliothèque de composants préconçus pour vous aider à construire rapidement votre site SaaS. Tous les composants sont construits avec TailwindCSS et sont entièrement personnalisables.

## Composants disponibles

Parcourez notre bibliothèque de composants pour trouver ce dont vous avez besoin :

- **[Sections Hero](/fr/docs/components/hero/)** - Sections hero accrocheuses pour vos pages d'atterrissage
- **[Tableaux de tarification](/fr/docs/components/pricing/)** - Beaux affichages de prix avec plusieurs styles
- **[Grilles de fonctionnalités](/fr/docs/components/features/)** - Présentez efficacement les fonctionnalités de votre produit

## Utilisation des composants

Les composants peuvent être utilisés dans vos fichiers de contenu en utilisant les shortcodes Hugo ou en créant des mises en page personnalisées. Chaque composant est conçu pour être :

- **Réactif** - Fonctionne sur toutes les tailles d'écran
- **Accessible** - Suit les directives WCAG
- **Personnalisable** - Facile à modifier les styles et le contenu

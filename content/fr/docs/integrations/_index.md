# Intégrations

Hugo Saasify s'intègre parfaitement avec les outils et services populaires pour étendre les fonctionnalités de votre site web.

## Intégrations disponibles

- **[Analytics](/fr/docs/integrations/analytics/)** - Suivez le comportement des visiteurs et les conversions
- **[Formulaires](/fr/docs/integrations/forms/)** - Collectez des prospects et des retours utilisateurs

## Configuration des intégrations

La plupart des intégrations peuvent être configurées dans votre fichier `hugo.toml` sous la section `[params]`. Chaque page d'intégration fournit des instructions de configuration spécifiques.

# Integraciones

Hugo Saasify se integra perfectamente con herramientas y servicios populares para extender la funcionalidad de tu sitio web.

## Integraciones disponibles

- **[Analytics](/es/docs/integrations/analytics/)** - Rastrea el comportamiento de visitantes y conversiones
- **[Formularios](/es/docs/integrations/forms/)** - Recopila leads y comentarios de usuarios

## Configuración de integraciones

La mayoría de las integraciones se pueden configurar en tu archivo `hugo.toml` bajo la sección `[params]`. Cada página de integración proporciona instrucciones de configuración específicas.

# Componentes

Hugo Saasify viene con una rica biblioteca de componentes preconstruidos para ayudarte a construir tu sitio SaaS rápidamente. Todos los componentes están construidos con TailwindCSS y son totalmente personalizables.

## Componentes disponibles

Explora nuestra biblioteca de componentes para encontrar lo que necesitas:

- **[Secciones Hero](/es/docs/components/hero/)** - Secciones hero llamativas para tus páginas de aterrizaje
- **[Tablas de precios](/es/docs/components/pricing/)** - Hermosas visualizaciones de precios con múltiples estilos
- **[Grillas de características](/es/docs/components/features/)** - Muestra las características de tu producto de manera efectiva

## Uso de componentes

Los componentes se pueden usar en tus archivos de contenido usando shortcodes de Hugo o creando layouts personalizados. Cada componente está diseñado para ser:

- **Responsivo** - Funciona en todos los tamaños de pantalla
- **Accesible** - Sigue las pautas WCAG
- **Personalizable** - Fácil de modificar estilos y contenido

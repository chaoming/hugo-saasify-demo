# Componentes

Hugo Saasify vem com uma rica biblioteca de componentes pré-construídos para ajudá-lo a construir seu site SaaS rapidamente. Todos os componentes são construídos com TailwindCSS e são totalmente personalizáveis.

## Componentes disponíveis

Navegue pela nossa biblioteca de componentes para encontrar o que você precisa:

- **[Seções Hero](/pt/docs/components/hero/)** - Seções hero atraentes para suas páginas de destino
- **[Tabelas de preços](/pt/docs/components/pricing/)** - Exibições de preços bonitas com múltiplos estilos
- **[Grades de recursos](/pt/docs/components/features/)** - Mostre os recursos do seu produto de forma eficaz

## Usando componentes

Os componentes podem ser usados em seus arquivos de conteúdo usando shortcodes do Hugo ou criando layouts personalizados. Cada componente é projetado para ser:

- **Responsivo** - Funciona em todos os tamanhos de tela
- **Acessível** - Segue as diretrizes WCAG
- **Personalizável** - Fácil de modificar estilos e conteúdo
